Security & Privacy
🔑
Per-binding secret key
Each bot binding generates a unique botSecret scoped to your device. Even if multiple users share the same bot, each bot session can only access the device it was bound to — using another user's credentials will result in a 403 error.
📋
Accessibility Tree only — no screenshots
The bot sees a structured list of UI elements (text, buttons, input fields) — not a screenshot of your screen. Sensitive visuals (images, videos, drawing canvas) are never captured.
🛑
You stay in control
Remote Control is off by default. You can disable it at any time in App Settings → Remote Control. Revoking the bot binding immediately invalidates its botSecret.
